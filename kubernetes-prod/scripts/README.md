# Скрипты для автоматизации

Эта папка содержит вспомогательные скрипты для выполнения домашнего задания.

## Скрипты

### create-vms.sh
Интерактивный скрипт для создания 4 ВМ в Yandex Cloud (1 master + 3 worker).

**Использование:**
```bash
./scripts/create-vms.sh
```

Скрипт запросит:
- Folder ID (можно оставить пустым для использования текущего)
- Zone (по умолчанию: ru-central1-a)
- Subnet ID (можно оставить пустым для автоматического поиска)
- Образ Ubuntu (можно оставить пустым для автоматического поиска)
- Путь к SSH ключу (по умолчанию: ~/.ssh/id_rsa.pub)

**Требования:**
- Настроенный YC CLI (`yc init`)
- SSH ключ для доступа к ВМ

### check-vm.sh
Скрипт для проверки состояния ВМ в Yandex Cloud.

**Использование:**
```bash
./scripts/check-vm.sh
```

### check-existing-cluster.sh
Скрипт для проверки существующего кластера Kubernetes на ВМ.

**Использование:**
```bash
./scripts/check-existing-cluster.sh <VM_IP_OR_NAME>
```

Проверяет:
- Установленные компоненты (kubeadm, kubectl, kubelet, containerd)
- Статус сервисов
- Состояние кластера (если развернут)
- Ресурсы ВМ

### create-vms-manual.md
Инструкция по ручному созданию ВМ через веб-консоль или команды yc.

## Альтернатива: Ручное создание

Если скрипты не работают, следуйте инструкциям в `create-vms-manual.md` или создайте ВМ через веб-консоль Yandex Cloud.
